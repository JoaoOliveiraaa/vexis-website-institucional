## Dicas para as Imagens

- Use imagens de alta qualidade
- Mantenha proporção 16:9 ou 1.91:1 (ideal para redes sociais)
- Formato JPG ou PNG
- Tamanho de arquivo: até 500KB (otimizado para web)
- Evite texto nas imagens (título já aparece no card)

## Onde Encontrar Imagens

- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)
